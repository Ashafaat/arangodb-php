Package: triagens\ArangoDb
Transaction object A transaction is an object that is used to prepare and send a transaction to the server.
The object encapsulates: * the collections definitions for locking * the actual javascript function * additional options like waitForSync, lockTimeout and params The transaction object requires the connection object and can be initialized with or without initial transaction configuration. Any configuration can be set and retrieved by the object's methods like this: $this->setAction('function (){your code};'); $this->setCollections(array('read' => 'my_read_collection, 'write' => array('col_1', 'col2'))); or like this: $this->action('function (){your code};'); $this->collections(array('read' => 'my_read_collection, 'write' => array('col_1', 'col2'))); There are also helper functions to set collections directly, based on their locking: $this->setWriteCollections($array or $string if single collection) $this->setReadCollections($array or $string if single collection)
