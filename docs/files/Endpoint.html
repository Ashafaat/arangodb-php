Endpoint specification
An endpoint contains the server location the client connects to the following endpoint types are currently supported (more to be added later): - tcp://host:port for tcp connections - unix://socket for UNIX sockets (provided the server supports this) - ssl://host:port for SSL connections (provided the server supports this) Note: SSL support is added in ArangoDB server 1.1
