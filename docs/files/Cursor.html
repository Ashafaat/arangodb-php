\triagens\ArangoDb\Cursor
jump to top

Package: triagens\ArangoDb
Provides access to the results of a read-only statement
The cursor might not contain all results in the beginning. If the result set is too big to be transferred in one go, the cursor might issue additional HTTP requests to fetch the remaining results from the server.
